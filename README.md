# Library Management System

## Project Description
This project is a simple Library Management System developed using C++.

The program allows the user to:
- Add books
- Store book information using struct
- Search for a book by ID
- Update book information
- Display updated data

---

## Features
- Add multiple books
- Search books using Book ID
- Update:
  - Book Name
  - Author Name
  - Number of Pages
  - Book Status
- Display updated information

---

## Concepts Used
This project uses the following C++ concepts:
- Variables
- Arrays
- Structures (struct)
- Loops
- If statements
- Boolean variables
- User Input and Output

---

## How to Run
1. Open the project in any C++ compiler.
2. Run the program.
3. Enter the number of books.
4. Enter book information.
5. Search for a book using Book ID.
6. Update the book information.
